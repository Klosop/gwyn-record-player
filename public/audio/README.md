# Audio files

Add the seven real MP3 files here as \`record-01.mp3\` through \`record-07.mp3\`.

Use MP3 rather than WAV to keep the Git repository and initial download small. About 192 kbps is a good quality/size balance. GitHub rejects individual Git objects larger than 100 MB, and the complete published Pages site should remain below 1 GB.

The repository and deployed songs are public on GitHub Pages.
